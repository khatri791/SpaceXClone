# SpaceXClone

This project is a front-end clone of the SpaceX website. It replicates the user interface and design of the SpaceX website, allowing users to explore
SpaceX's missions, rockets, and other information.


## Features

- Fully Responsive
- Live previews
- Fullscreen mode
- Cross platform
